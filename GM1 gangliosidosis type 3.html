    • GM1 gangliosidosis type 3¶

    GM1 神经节苷脂沉积症 3 型

    疾病信息汇总

      疾病基本信息 主要临床表现 疾病特征信息 检查与诊断 治疗方案 遗传咨询

    疾病基本信息¶

    • 疾病其他中文别名： 神经节苷脂病 GM1 3 型；β-半乳糖苷酶缺乏症 3 型；成人 GM1 神经节苷脂沉积症
    • 疾病其他英文别名： Gangliosidosis GM1 type 3; Beta-galactosidase deficiency type 3; Adult GM1 gangliosidosis
    • Orpha Number： 79257
    • OMIM Number： 230650
    • 疾病简介： 是一种遗传性溶酶体贮积症，病因为GLB1基因变异引起溶酶体β-半乳糖苷酶缺乏，呈现常染色体隐性遗传。成人发病和小脑功能障碍是本病的主要特征。该疾病迄今尚无有效治疗方法。
    • 致病基因名称： GLB1(galactosidase beta 1，β-半乳糖苷酶)
    • 致病基因基因位置： 3p22.3
    • 致病基因基因描述： GLB1基因编码的β-半乳糖苷酶是糖基水解酶 35 蛋白家族的成员。选择性剪接产生多个转录本，其中至少一个编码前体蛋白，后者经过蛋白水解加工后形成成熟的溶酶体酶。该酶水解来自神经节苷脂底物和其它糖复合物的末端 β 半乳​​糖
    • 致病基因与其他疾病的关系： 无
    • 其他：

    主要临床表现¶

    • 综合描述： 言语不清；构音障碍；肌张力障碍；身材矮小；轻度脊椎异常； 血液和造血异常；面部异常；弥漫性脑萎缩；轻度智力残疾；缓慢进行性痴呆；共济失调；
    • HPO分类异常：
      • 头部和颈部的异常:面部异常
      • 神经系统异常:弥漫性脑萎缩；轻度智力残疾；缓慢进行性痴呆；共济失调；
      • 心血管系统异常: 血液和造血异常
      • 声音异常:言语不清；构音障碍
      • 生长异常:身材矮小
      • 骨骼系统异常:轻度脊椎异常
      • 肌肉组织异常:肌张力障碍

    疾病特征信息¶

    • 流行病学： 小于百万分之一
    • 患病率： 1e-06
    • 遗传模式： 常染色体隐形遗传
    • 疾病特征： 成人发病；小脑功能障碍
    • 其他：

    检查与诊断¶

    • 实验室检查指标：
    • 主要实验室检查异常指标：
      • 尿常规-检测结果:尿液中可见硫酸角质素排出
      • 生化检查-碱性磷酸酶(ALP):升高
    • 基因筛查-检测结果： GLB1基因突变
    • 其他检查：
    • 鉴别诊断： 应与粘多糖贮积症、鞘脂贮积症和寡糖贮积症相区分
    • 自然病史： 无
    • 并发症： 肺炎
    • 预后： 无

    遗传咨询¶

    • 基因诊断综合描述： 如果临床怀疑该病，或者怀疑遗传代谢病，应作GLB1基因或全外显子组（WES）检测；发现有GLB1基因编码区有致病变异或致病的拷贝数变异（CNV）即可确诊；如是可疑变异或意义未明变异，可应用生物信息学手段进行变异性质预测，有条件的实验室应进行细胞层面的功能学研究。如果未发现变异，应继续全基因组检测，确定非编码区域是否存在变异。
    • 基因诊断检查方法： WES；CNV；WGS
    • 先证者家族亲属发病风险评估： 确定先证者的致病基因型后，应检测其父母相应位点的基因型，确定变异来源；对家族中的其他亲属，在知情同意的前提下，亦应进行GLB1基因测序，然后进行发病风险评估。
    • 再次生育风险评估： 如果先证者父母希望继续生育下一胎，首先建议选用辅助生殖的方式受孕。如果只选择自然受孕，则应作产前诊断，确定胎儿是否存在GLB1基因变异。胎儿出生后，应再次进行GLB1基因检测。
    • 辅助生殖评估： 对受精卵进行GLB1基因检测。选择无GLB1基因变异的受精卵进行子宫内植入。同上进行产前诊断和新生儿基因检测。
    • 婚姻建议： 凡携带GLB1基因变异者，不宜与同样携带GLB1基因变异者生育。如果必须生育，则参照再次生育风险评估和辅助生殖风险评估。
    • 干预或治疗建议： 目前没有针对该病的基因治疗、细胞治疗和酶替代治疗。可根据实验室检测结果，参照治疗方案进行对症治疗。
    • 预防： 产前诊断
