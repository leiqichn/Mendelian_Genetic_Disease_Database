    • Mitochondrial complex III deficiency¶

    线粒体复合物III缺乏症

    疾病信息汇总

      疾病基本信息 主要临床表现 疾病特征信息 检查与诊断 治疗方案 遗传咨询

    疾病基本信息¶

    • 疾病其他中文别名： 单纯型线粒体呼吸链复合物III缺乏症；单纯型线粒体细胞色素C还原酶缺乏症；单纯型辅酶Q-细胞色素C还原酶缺乏症
    • 疾病其他英文别名： Isolated CoQ-cytochrome C reductase deficiency; Isolated coenzyme Q-cytochrome C reductase deficiency; Isolated mitochondrial respiratory chain complex III deficiency; Isolated ubiquinone-cytochrome C reductase deficiency
    • Orpha Number： 1460
    • OMIM Number： 124000;615157;615839;615160;615453;615159;616111;615824;618775;615158
    • 疾病简介： 线粒体复合物III缺陷是较罕见的线粒体疾病之一，由De Lonlay等人在2001年首次报道，发病率<1 / 1000000。病因为BCS1L; TTC19; UQCRB; UQCRQ; UQCRC2; CYC1; UQCC2; LYRM7等基因变异引起线粒体复合物III缺乏，呈现常染色体隐性遗传或线粒体遗传。患者出生时即出现乳酸中毒、肌张力低下、低血糖、发育不良、脑病和精神运动发育延迟是本病的特征。目前全世界报道了10多例患者，无中国病例报道。主要治疗方式是维生素K2及维生素C等的补充治疗。
    • 致病基因名称： BCS1L（BC1 (Ubiquinol-Cytochrome C Reductase) Synthesis-Like， BC1(泛素-细胞色素C还原酶)合成类似物）; TTC19（Tetratricopeptide Repeat Domain 19， 四三肽重复结构域19）; UQCRB（Ubiquinol-Cytochrome C Reductase Binding Protein， 泛素细胞色素C还原酶结合蛋白）; UQCRQ（Ubiquinol-Cytochrome C Reductase Complex III Subunit VII， 泛素-细胞色素C还原酶复合物III亚基VII）; UQCRC2 （Ubiquinol-Cytochrome C Reductase Core Protein 2， 细胞色素C还原酶核心蛋白2）; CYC1 （Cytochrome C1， 细胞色素C₁）; UQCC2 （Ubiquinol-Cytochrome C Reductase Complex Assembly Factor 2， 泛素-细胞色素C还原酶复合物组装因子2）; LYRM7（LYR Motif Containing 7）;
    • 致病基因基因位置： 2q35; 17p12; 8q22.1; 5q31.1; 16p12.2; 8q24.3; 6p21.31; 5q23.3-q31.1; 11q12.3; 19q12
    • 致病基因基因描述： BCS1L; TTC19; UQCRB; UQCRQ; UQCRC2; CYC1; UQCC2; LYRM7等基因编码线粒体复合物III亚基。线粒体复合物III是电子传递链上的核心元素，它介导辅酶Q的氧化和细胞色素C的还原，通过一系列的氧化还原过程最终形成ATP，为机体组织提供能量。
    • 致病基因与其他疾病的关系： GRACILE综合征
    • 其他：

    主要临床表现¶

    • 综合描述： （1）精神发育迟滞 （2）喂养困难 （3）代谢性酸中毒 （4）肝损害 （5）肾脏损害
    • HPO分类异常：
      • 头部和颈部的异常:头发干燥；小头畸形
      • 耳部异常:耳聋
      • 神经系统异常:智力低下；线粒体脑病；反射亢进；痉挛；运动功能紊乱；癫痫；大脑萎缩；小脑萎缩；脑白质病变
      • 心血管系统异常:心肌肥大
      • 生长异常:精神运动发育迟缓
      • 消化系统异常:肝纤维化；胆汁淤积症；胆囊炎；肝脏铁沉积；肝小泡脂肪变性；喂养困难
      • 全身性症状:生长发育停滞；出生后发育不良
      • 代谢紊乱/稳态失衡:乳酸性酸中毒；代谢性酸中毒；低血糖；进食后高血糖
      • 眼部异常:白内障；失明；上睑下垂
      • 胎儿产前发育或出生异常:出生前发育不良；
      • 血液和造血组织异常:肝功能减退导致的凝血障碍
      • 肌肉组织异常:肌张力低下；肌肉无力；
      • 泌尿生殖系统异常:肾小管间质性肾炎；肾小管病变

    疾病特征信息¶

    • 流行病学： 单纯性复合物II缺陷导致线粒体疾病的情况较为罕见，占所有线粒体呼吸链缺陷疾病的不到2%.
    • 患病率： <1 / 1000000
    • 遗传模式： 常染色体隐形遗传；线粒体遗传
    • 疾病特征： 出生时即出现乳酸中毒、肌张力低下、低血糖、发育不良、脑病和精神运动发育延迟。

    • 实验室检查指标：

      • 生化检查-葡萄糖（Glu）:降低

    • 鉴别诊断： GRACILE综合征；布加氏综合征（Björnstad综合征）
    • 自然病史： 如果不治疗，会引起[代谢性酸中毒]，[高血糖]或[低血糖]

    • 使用药物及其反应： 治疗主要以改善线粒体功能为主，大剂量的维生素K3、维生素C被证明能够改善症状。癫痫发作患者需要使用抗癫痫药物如扑痫酮。

    遗传咨询¶

    • 基因诊断综合描述： 如果临床怀疑该病，或者怀疑遗传代谢病，应作BCS1、LTTC19、UQCRB、UQCRQ、UQCRC2、CYC1、UQCC2和LYRM7等基因或全外显子组（WES）检测；发现有BCS1L等基因编码区有致病变异或致病的拷贝数变异（CNV）即可确诊；如是可疑变异或意义未明变异，可应用生物信息学手段进行变异性质预测，有条件的实验室应进行细胞层面的功能学研究。如果未发现变异，应继续全基因组检测，确定非编码区域是否存在变异。
    • 基因诊断检查方法： WES；CNV；WGS
    • 先证者家族亲属发病风险评估： 确定先证者的致病基因型后，应检测其父母相应位点的基因型，确定变异来源；对家族中的其他亲属，在知情同意的前提下，亦应进行BCS1L等基因测序，然后进行发病风险评估。
    • 再次生育风险评估： 如果先证者父母希望继续生育下一胎，首先建议选用辅助生殖的方式受孕。如果只选择自然受孕，则应作产前诊断，确定胎儿是否存在BCS1L等基因变异。胎儿出生后，应再次进行BCS1L等基因检测，同时检测相应线粒体酶水平。
    • 辅助生殖评估： 对受精卵进行BCS1L等基因检测。选择无BCS1L等基因变异的受精卵进行子宫内植入。同上进行产前诊断和新生儿基因检测。
    • 婚姻建议： 凡携带BCS1L等基因变异者，不宜与同样携带BCS1L等基因变异者生育。如果必须生育，则参照再次生育风险评估和辅助生殖风险评估。
    • 干预或治疗建议： 目前没有针对该病的基因治疗、细胞治疗和酶替代治疗。可根据实验室检测结果，参照治疗方案进行对症治疗。
    • 预防： 产前诊断
